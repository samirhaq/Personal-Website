About Me

A little bit about my story so far.

  • 1998-2015

    Humble Beginnings

    I was born in Mississauga, Canada to two Bengali parents. I've always had a love for music, movies, sports and especially computers.

  • June 2016

    Graduating Highschool

    I graduated Meadowvale Secondary School with an 88% average and a new found love for developing software thanks to our wonderful computer science teacher Ms. Ghanekar who gave me the base for programming that I have built all of my skills off of today. She helped me find a new passion for it where I would even work on projects myself. In grade 12, a friend of mine and I built a "Who Wants to be a Millionaire" game using Python from scratch. I finished with a 93% and 91% in her classes in grade 11 and 12, leading me to the pursue a career in that field.

  • September 2016

    Started at the Univeristy of Guelph

    I owe the University of Guelph for much of what I have learned. At my time at the Univeristy so far I have learned many valuable aspects of computing and software development. I took many valuable classes that have taught me things such as Object-Oriented Programming, Data Structures, Efficient Algortihtms, Operating Systems and Full Stack Web Development.

  • May 2018

    Started Job at Kenna

    For my first term I worked at Kenna, a marketing firm. Here I gained admirable knowledge in databases. Working with a lot of SQL, I worked on queries to pull out information from different tables to create data reports for clients. A lot of my work there invloved learning and understanding SQL, and at times even teaching it to my co-workers. Some of my tasks involved writing queries to QA reports, building an ER diagram for a procedure that creates a large table from many smaller base tables, and running SQL training sessions for others wanting to learn more about how to use SQL.

VendorX

(In Progress)

An E-commerce app integrated with Shopify. The app is to simplify communications with vendors so that the process as automated as possible. Many stores never hold any inventory themselves, rather take an order from a customer, and then send that order to the vendors. This app will automate the process so as soon as an order is issued from a customer. The order will be parsed and exported into a csv file for each vendor, which will then be sent out to the specific products for each vendor.

Uses:

  • Ruby
  • Ruby on Rails
  • AJAX
  • JavaScript
  • RESTful API
  • Oauth2
  • HTML
  • CSS

Geanology App

(In Progress)

A school assignment and my first time developing a full stack application. The app lets you upload a GEDCOM file, and parses the information using a backend developed in C, and stores it in a database using SQL. You can then see the family tree information such as a specific members descendants or anscestors, the number of members in a family, the number of members in a generation etc.

Uses:

  • C
  • Linked List
  • Hash Table
  • JQuery
  • AJAX
  • SQL
  • NodeJS
  • HTML
  • Bootstrap
  • CSS

Personal Assistant

(In Progress)

A text-based (possibly devoloping a UI) school assignment using a BTS to create a self-learning personal assistant. You can ask the assistant any question you would like and it will give you an answer that best fits the criteria and whatever knowledge the assistant has on the topic. You can then rate how helpful the answer was in order for the assistant to gather knowledge so it can help you get a better answer everytime. You can also enter your own custom questions, and answers to those questions.

Uses:

  • C
  • JQuery
  • Binary Search Tree
